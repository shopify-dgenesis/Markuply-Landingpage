FAQ

Questions, answered

Do I need to know how to code?

No. You paste HTML the AI wrote and make every decision through visual controls — pickers, buttons, and previews. You never edit code by hand (though the clean output is there if you want it).

Which AI tools does it work with?

Any tool that outputs HTML — ChatGPT, Claude, Gemini, and Grok are what it's tuned for, and the built-in prompt generator writes an optimized prompt for each of them.

Will it affect my theme?

No. Generated CSS is scoped inside a unique wrapper, so the page's styles physically can't leak into your theme — and your theme's styles are reset inside the page block.

What exactly is free?

Three full page conversions, forever — the complete detect → map → brand → review flow, with no card and no expiry. Pro ($18.99/month) removes the cap and unlocks the power features: working forms, theme match, product mapping, AI alt text, mobile auto-fixes, and unlimited version history.

What if I publish something and regret it?

Every publish is snapshotted to version history. You can preview any earlier version and roll the live page back to it with one click.

Is the AI-generated HTML safe to put on my store?

Markuply strips scripts, inline event handlers, and unsafe embeds automatically before anything is generated — and shows you a report of exactly what was removed.
